# Contributing

This repository is a **personal portfolio** maintained by Jerico G. Sison.

- **Bug reports** or **accessibility** suggestions aligned with the site’s goals are welcome via GitHub issues or direct contact (see the live site).
- **Large unsolicited redesigns** or dependency-heavy refactors are unlikely to be merged without prior discussion.
- If you **fork** the project under the [MIT License](./LICENSE), replace biography text, imagery, and identifiers so the deployed site clearly reflects **your** work.

Thank you for respecting licence terms and for keeping feedback constructive and specific.
