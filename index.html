Flagship build

Capstone

AI-Assisted Admission System at PSAU

Handles the full admission lifecycle: registration, application submission, verification, exam scheduling, scoring, course assignment, enrollment scheduling, and completion. Implemented in PHP with server-rendered pages and JavaScript modules; MySQL stores records; Firebase-hosted email powers OTP (email-based flows in current code, not SMS); AI assistance includes chatbot and recommendation proxies under public/ai/.

My role

Developer on the capstone, contributing across applicant-facing (public/) and administration (admin/) areas, shared PHP utilities (includes/), Firebase email wiring (firebase/), and alignment with deployment tooling (Render, Docker-related configs).

Tech stack

PHP ≥7.4 MySQL JavaScript HTML/CSS Firebase email reCAPTCHA Composer

Challenges & how they were addressed

  • Secure configuration across environments: database settings driven by environment variables (DB_*) with sensible separation from production secrets (per repo deployment docs).
  • Reliable applicant verification: email OTP endpoints (send_email_otp.php, admin equivalents), OTP expiry and rate-limit helpers in shared includes, plus hashed passwords and session auth.
  • Integrating AI features safely: chatbot/recommendation traffic routed through proxy endpoints instead of exposing backend logic directly to the browser.
