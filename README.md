# Jerico Sison Portfolio

An attractive personal portfolio website built using **HTML, CSS, and JavaScript only**.

## Live Project Structure

This repository contains the portfolio files directly at the root:

- `index.html`
- `styles.css`
- `script.js`
- `img/` (image assets including profile photo)

## Features

- Modern responsive design
- Smooth section animations and reveal effects
- Custom mouse cursor design with hover highlight interactions
- Animated loader when the site opens
- Projects section showcasing real work (including PSAU Admission System)
- Contact form that opens the default email client

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Highlight

### PSAU Admission System 2.0

Main portfolio-highlighted system project:
- Repository: [psau-admission-system2](https://github.com/jerico-sison/psau-admission-system2)
- Focus: secure admission workflow with applicant and admin interfaces
- Stack includes PHP, MySQL, JavaScript, and Firebase services

## How to Run Locally

### Option 1: Open directly
1. Download/clone this repository.
2. Open `index.html` in your browser.

### Option 2: Use XAMPP
1. Put this project inside your `htdocs` folder.
2. Start Apache in XAMPP.
3. Open:
   - `http://localhost/Porfolio/`

## Customization Guide

- Update personal information in `index.html`
- Adjust colors and visual style in `styles.css`
- Edit animations/interactions in `script.js`
- Replace profile image inside `img/`
